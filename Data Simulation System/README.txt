The software was developed to configure raspberry pi through a ethernet cable.Once the pi is connected to the pc the software
calls the pi through its hostname and configures things like date,wifi credentials,etc.This software was designed for a firm 
so its source files can't be disclosed.

The project has the following sub folders i.e 

1.Screenshots - The folder has images of the software.The software has been designed using python tkinter which configures the date,details of sensors and network credentials.The software runs on the pc and the pi is connected to the pc through the ethernet cable.
