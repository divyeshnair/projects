The project is all about developing an application for the ease of people to make transportation easier and comfortable. In this project a navigation system was 
developed using a gps module connected to certain hardware, which would update the location of the vehicle in the android application. The latitude and longitude 
parameters obtained from the gps module were projected to the Google map which helped people track the vehicles at real life scenario.
