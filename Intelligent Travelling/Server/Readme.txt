*This folder has python-flask server code which was developed to host the google map.
*This server would simultaneously accept the post requests obtained by the pi.
*Finally the obtained coordinates are projected to the google map with a marker as an indicator.


*Requirements
->Python 2.7.9
->pip install flask

*To launch the server, the file 'start.py' has to be run.Point the browser to the ip mentioned in the command line.
